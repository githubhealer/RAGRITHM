# RAGRITHM

## Setup

**Create a Virtual Env**

```bash
$ python -m venv venv
```

**Switch to virtualenv**

- Linux / OSX

```bash
$ source venv/bin/activate
```

- Windows

```cmd
venv\Scripts\activate.bat
```

**Install Packages**

```bash
$ pip install -r requirements.txt
```

**Run FastAPI**

```bash
$ fastapi dev main.py
```
