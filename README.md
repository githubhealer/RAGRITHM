# RAGRITHM

## Setup

**Create a Virtual Env**

```bash
$ python -m virtualenv .env
```

**Switch to virtualenv**

- Linux / OSX

```bash
$ source .env/scripts/activate
```

- Windows

```cmd
.env\Scripts\activate.bat
```

**Install Packages**

```bash
$ pip install -r requirements.txt
```

**Run FastAPI**

```bash
$ fastapi dev main.py
```
